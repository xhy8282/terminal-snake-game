# Terminal Snake Game

一个运行在命令行终端的经典贪吃蛇游戏，使用 Python 实现。

## 项目信息

- 最后修改：2023年8月6日
- 状态：可正常运行
- 用途：Python 教学示例项目

## 游戏特性

- 🎮 经典贪吃蛇玩法
- 🖥️ 纯命令行界面
- ⚡ 多线程渲染引擎
- 📊 使用 NumPy 处理游戏场景数据
- ⌨️ WASD 键盘控制

## 环境要求

```bash
pip install numpy keyboard
```

**注意**：`keyboard` 库在某些系统上可能需要管理员/root 权限。

## 运行方式

```bash
python 复杂1_贪吃蛇.py
```

### Windows 用户
确保以管理员身份运行命令提示符或 PowerShell。

### Linux/Mac 用户
```bash
sudo python3 复杂1_贪吃蛇.py
```

## 游戏操作

- `W` - 向上移动
- `S` - 向下移动
- `A` - 向左移动
- `D` - 向右移动

## 游戏规则

- 控制 `#`（蛇头）吃 `*`（豆子）
- 吃到豆子后蛇身 `@` 会变长
- 撞墙或咬到自己即游戏结束
- 场景大小可通过 `accuracy` 变量调整

## 技术实现

### 核心技术栈
- **NumPy**：场景数据矩阵处理
- **Threading**：多线程渲染刷新
- **Keyboard**：键盘事件监听
- **OS/Sys**：系统调用与控制台操作

### 代码结构
```
游戏场景矩阵 (display_data)
├── 0: 空地 '-'
├── 1: 蛇身 '@'
├── 2: 蛇头 '#'
└── 3: 豆子 '*'
```

### 关键机制
- **移动逻辑**：坐标列表头部插入，尾部删除
- **碰撞检测**：边界检测 + 自身坐标重复检测
- **随机生成**：豆子在空白区域随机刷新
- **防回头**：方向切换时阻止 180° 转向

## 配置参数

```python
accuracy = 500  # 场景精度（影响游戏区域大小）
fps = 10        # 刷新率（数值越大游戏越快）
```
